# i-hate-mental-health

A mental health app so generic it becomes avant-garde.

Built with stakeholder alignment, evidence-informed vibes, and no measurable joy.

## Run App

Apple dev cert cost too much.

No App Store. No TestFlight. No certificate pain.

Want app?

Clone repo. Open in Xcode. Pick simulator. Press Cmd+R.

Real device maybe need Apple account. Simulator easier. Use simulator.

```bash
git clone https://github.com/zhu-adam/i-hate-mental-health.git
cd i-hate-mental-health
open MentalHealth/MentalHealth.xcodeproj
```

Then Xcode open.

Pick iPhone simulator.

Press Cmd+R.

App run.

Stone tools sufficient.
